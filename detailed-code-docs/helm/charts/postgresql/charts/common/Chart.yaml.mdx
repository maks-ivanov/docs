## High-level description
This file is a Helm Chart.yaml for a common library chart used in Bitnami charts. It defines metadata and configuration for a reusable set of Helm chart components that are not independently deployable but serve as a foundation for other Bitnami charts.

### Chart Metadata
#### Description
This Chart.yaml file defines the metadata for a Helm chart named "common". It is a library chart that provides shared logic and templates for other Bitnami charts.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| annotations | object | Additional metadata about the chart |
| apiVersion | string | The version of the Chart.yaml schema |
| appVersion | string | The version of the application this chart represents |
| description | string | A brief description of the chart's purpose |
| home | string | The URL of the project's homepage |
| icon | string | The URL of the chart's icon |
| keywords | array | List of keywords describing the chart |
| maintainers | array | List of maintainers with their names and URLs |
| name | string | The name of the chart |
| sources | array | List of URLs to the chart's source code |
| type | string | The type of chart (library in this case) |
| version | string | The version of the chart itself |

#### Outputs
This file does not produce any outputs as it is a declarative configuration file.
