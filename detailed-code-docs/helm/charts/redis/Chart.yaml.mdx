## High-level description
This Chart.yaml file defines the metadata and configuration for a Helm chart that deploys Redis, an open-source, advanced key-value store. It specifies the chart version, app version, dependencies, and other essential information for deploying Redis using Helm.

| Dependency | Purpose |
|:-----------|:--------|
| common | Bitnami common chart, version 2.x.x |

This Chart.yaml file defines a Helm chart for Redis version 7.2.4, with a chart version of 18.9.0. It includes metadata about the chart, such as its description, home page, icon, and keywords. The chart depends on a common Bitnami chart and specifies four Docker images used in the deployment. The maintainer is listed as VMware, Inc., and the chart is licensed under Apache-2.0.
