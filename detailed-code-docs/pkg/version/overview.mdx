Here's a detailed documentation for the `pkg/version/version.go` file:

## High-level description
This file defines version-related functionality for the gptscript project. It provides structures and functions to handle version information, including tag, commit, and dirty state of the build.

## Code Structure
The code defines a `Version` struct and associated functions to create, manipulate, and represent version information. It also includes utility functions to retrieve Git commit information.
