## High-level description
This file, `diesel.toml`, configures the Diesel CLI tool for a Rust project. Specifically, it instructs Diesel where to output the generated SQL schema file.

## Table of contents

## Symbols
### Symbol Name
#### `[print_schema]`
##### Description
This section configures the behavior of Diesel's `print_schema` command, which generates a Rust module representing the database schema.

##### Inputs
This section takes configuration options as key-value pairs.

##### Outputs
This section itself doesn't produce output. However, the configuration it provides affects the output of the `print_schema` command.
