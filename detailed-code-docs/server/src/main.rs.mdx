## High-level description
This is the entry point for the Trieve server application. It delegates the main execution to the `trieve_server` module's `main` function.

## Table of contents
- Main function

## Symbols

### `main`
#### Description
The `main` function is the entry point of the Rust program. It calls the `main` function from the `trieve_server` module and returns its result.

#### Inputs
This function takes no inputs.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| Result | `std::io::Result&lt;()&gt;` | The result of the `trieve_server::main()` function, which is a `Result` type that either contains nothing (`()`) on success or an I/O error on failure. |

#### Internal Logic
1. Call `trieve_server::main()`
2. Return the result of the call
