## High-level description
This TOML configuration file defines connection parameters for a ClickHouse database. It specifies the URL, user credentials, and the default database to be used for ClickHouse migrations.

## Table of contents
- ClickHouse connection configuration

## Symbols

### `url`
#### Description
Specifies the URL of the ClickHouse server.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| url | string | The URL of the ClickHouse server |

### `user`
#### Description
Defines the username for authenticating with the ClickHouse server.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| user | string | The username for ClickHouse authentication |

### `password`
#### Description
Specifies the password for the ClickHouse user.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| password | string | The password for ClickHouse authentication |

### `database`
#### Description
Defines the default database to be used for ClickHouse operations.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| database | string | The name of the default ClickHouse database |

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
| url | string | "http://localhost:8123" | The URL of the ClickHouse server |
| user | string | "clickhouse" | The username for ClickHouse authentication |
| password | string | "password" | The password for ClickHouse authentication |
| database | string | "default" | The name of the default ClickHouse database |
