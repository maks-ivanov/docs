## High-level description

The `NavigationMenuPlugin.tsx` file defines a React component plugin for rendering a navigation menu. This plugin is part of a larger plugin architecture and is designed to be stateless, meaning it does not manage its own state but instead relies on data passed to it. The navigation menu can be configured to display items either horizontally or vertically, and it supports both individual menu items and grouped items with optional tooltips.

## Code Structure

The main symbols in the code are interconnected as follows:

- `NavigationMenuPlugin`: A class implementing the `IStatelessPlugin` interface, responsible for rendering the navigation menu using the `NavMenuComponent`.
- `NavMenuComponent`: A functional component that takes menu data and orientation as props and renders the menu items.
- `ListItem`: A functional component used to render individual menu items, potentially with tooltips.

The `NavigationMenuPlugin` class uses the `NavMenuComponent` to render the menu, and `NavMenuComponent` uses `ListItem` to render each menu item.
