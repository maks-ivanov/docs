## High-level description

The `TextAreaPlugin.tsx` file defines a plugin for rendering a customizable text area component in a React application. It implements the `IPlugin` interface, providing validation for the text area properties using the `zod` library and rendering the component with optional features like labels, maximum length indicators, and full-width display.

## Code Structure

- **TextAreaPlugin**: A class implementing the `IPlugin` interface, responsible for defining the text area component's properties, validation, and rendering logic.
- **TextAreaComponent**: A functional component that renders the text area with additional features like a label and a character count indicator.

## Symbols

### `TextAreaPlugin`
#### Description
The `TextAreaPlugin` class implements the `IPlugin` interface to provide a text area component with customizable properties such as placeholder, label, max/min length, disabled state, number of rows, and full-width option. It uses the `zod` library for validating these properties and renders the component using the `TextAreaComponent`.

#### Internal Logic
- **Validator**: Uses `zod` to define a schema for validating the text area properties.
- **Render Method**: Calls the `TextAreaComponent` with the necessary props to render the text area.

### `TextAreaComponent`
#### Description
A functional component that renders a text area with optional features such as a label and a character count indicator. It uses the `Textarea` component from the UI library and the `Labeled` component for displaying labels.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | `TextAreaComponentProps` | The properties for the text area, including value, setValue, and other display options. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| output | `JSX.Element` | The rendered text area with optional label and character count. |

#### Internal Logic
- **Bottom Adornment**: Displays a character count if `maxLength` is specified.
- **Labeled Component**: Wraps the `Textarea` component with a label if provided.
- **Textarea Component**: Renders the actual text area with properties like rows, maxLength, minLength, and placeholder.
