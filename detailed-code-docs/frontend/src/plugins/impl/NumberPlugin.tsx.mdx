## High-level description

The `NumberPlugin.tsx` file defines a React component plugin called `NumberPlugin` that is used to render a number input field with additional features such as debouncing, labeling, and full-width display options. It utilizes the `zod` library for data validation and integrates with other components and hooks to provide a customizable and interactive number input experience.

## Code Structure

- **NumberPlugin**: A class implementing the `IPlugin` interface, responsible for rendering the number input component and validating its data.
- **NumberComponent**: A functional React component that renders the number input field with optional debouncing and labeling.
