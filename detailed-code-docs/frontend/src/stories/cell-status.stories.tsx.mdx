## High-level description

The target file `cell-status.stories.tsx` is a Storybook configuration file that defines various stories for the `CellStatusComponent`. This component is used to display the status of a cell in an editor, with different visual states such as idle, edited, interrupted, running, queued, and disabled. The stories provide different configurations of the component to demonstrate its appearance and behavior under various conditions.

## Code Structure

The main symbols in the code are interconnected as follows:
- `CellStatusComponent`: The primary React component being showcased in the stories.
- `CellStatusComponentProps`: The type definition for the props that `CellStatusComponent` accepts.
- `TooltipProvider`: A wrapper component used to provide tooltip functionality to the `CellStatusComponent`.
- `Template`: A function that serves as a base for creating different stories by binding specific arguments to the `CellStatusComponent`.
