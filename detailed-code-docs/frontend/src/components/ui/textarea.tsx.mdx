## High-level description

The `textarea.tsx` file defines a React component named `Textarea`, which is a customizable text area input field. It extends the standard HTML `&lt;textarea&gt;` element with additional styling and functionality, such as the ability to include a bottom adornment and handle click events to stop event propagation.

## Code Structure

The main symbol in this file is the `Textarea` component, which is a functional component using `React.forwardRef` to allow for a ref to be passed to the underlying `&lt;textarea&gt;` element. The component accepts props that extend the standard HTML textarea attributes and adds a custom `bottomAdornment` prop. The `cn` utility is used for conditional class name merging, and the `Events` utility is used to handle event propagation.

### `Textarea`
#### Description
The `Textarea` component is a React component that renders a styled `&lt;textarea&gt;` element with optional bottom adornment. It is designed to be flexible and reusable, allowing for additional props and styles to be passed in.

#### Internal Logic
- The component uses `React.forwardRef` to forward the ref to the `&lt;textarea&gt;` element.
- It applies a set of default styles using the `cn` utility to merge class names, allowing for hover, focus, and disabled states.
- The `onClick` event handler is set to `Events.stopPropagation()` to prevent click events from propagating beyond the textarea.
- If `bottomAdornment` is provided, it is rendered within a `div` positioned absolutely at the bottom right of the textarea.
