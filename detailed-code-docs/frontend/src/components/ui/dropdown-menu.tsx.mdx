## High-level description

The `dropdown-menu.tsx` file defines a set of React components that provide a customizable dropdown menu interface using the Radix UI library. These components are designed to be flexible and reusable, allowing developers to create complex dropdown menus with various items, submenus, and interactive elements like checkboxes and radio buttons. The file leverages Radix UI primitives and extends them with additional styling and functionality.

## Code Structure

The main symbols in the code are interconnected as follows:
- **DropdownMenu Components**: These are wrappers around Radix UI primitives, providing additional styling and functionality.
- **Styling Variants**: The components use styling variants imported from `menu-items.tsx` to apply consistent styles across different menu elements.
- **Icons**: Icons from the `lucide-react` library are used to enhance the visual representation of menu items.
- **Utility Functions**: The `cn` utility function is used to merge class names for styling purposes.
