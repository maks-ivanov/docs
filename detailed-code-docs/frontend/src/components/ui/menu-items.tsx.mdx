## High-level description

The target file, `menu-items.tsx`, defines a set of styled components and utility functions for creating and managing menu items in a React application. It utilizes the `class-variance-authority` library to manage CSS class variants, allowing for flexible styling of menu components such as menu items, labels, separators, and shortcuts. These components are designed to be used in various UI elements like dropdowns, context menus, and command menus.

## Code Structure

The file primarily consists of styled component definitions using the `cva` function from the `class-variance-authority` library. Each component is defined with a base set of CSS classes and optional variants that modify the styling based on certain conditions. The `MenuShortcut` component is a React functional component that renders a span element with specific styling.
