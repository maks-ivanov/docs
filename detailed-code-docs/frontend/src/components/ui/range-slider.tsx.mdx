## High-level description

The `RangeSlider` component in the target file is a React component that provides a customizable range slider UI element. It leverages the `@radix-ui/react-slider` library to create a slider with two thumbs, allowing users to select a range of values. The component also integrates tooltips to display the current values of the slider thumbs, formatted using a custom number formatting function.

## Code Structure

The `RangeSlider` component is built using several interconnected symbols:

- **`SliderPrimitive`**: A set of components from the `@radix-ui/react-slider` library that provide the basic slider functionality.
- **`Tooltip` components**: Used to display tooltips for the slider thumbs, providing additional information about the current values.
- **`useBoolean` hook**: Manages the open state of the tooltips, allowing them to be shown or hidden based on user interactions.
- **`prettyScientificNumber` function**: Formats the slider values for display in the tooltips.

## Symbols

### `RangeSlider`
#### Description
The `RangeSlider` component is a forward-ref React component that renders a range slider with two thumbs. It allows users to select a range of values and displays tooltips with formatted values when the thumbs are focused or hovered over.

#### Inputs
| Name     | Type   | Description                                                                 |
|:---------|:-------|:----------------------------------------------------------------------------|
| className| string | Additional CSS classes to apply to the slider root element.                 |
| valueMap | function | A function that maps the slider's internal value to a display value.      |
| props    | object | Additional props passed to the `SliderPrimitive.Root` component.            |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| N/A  | N/A  | The component does not return any values but renders UI elements. |

#### Internal Logic
- The component uses the `useBoolean` hook to manage the open state of the tooltips.
- It renders a `SliderPrimitive.Root` component with a track and two thumbs.
- Each thumb is wrapped in a `TooltipTrigger` and `TooltipRoot` to display a tooltip when focused or hovered.
- The tooltips display values formatted by the `prettyScientificNumber` function, using the `valueMap` function to convert slider values.

## Side Effects
- The component modifies the tooltip's open state based on user interactions (focus, blur, mouse enter, and mouse leave events).
