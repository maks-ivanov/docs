## High-level description

The `debugger-code.tsx` file defines a React component named `Debugger` that provides a user interface for a code debugging tool. It includes components for displaying code output, accepting code input, and providing control buttons for debugging actions such as executing the next line, continuing execution, printing the stack trace, and accessing help.

## Code Structure

The main symbols in the code are interconnected as follows:
- `Debugger`: The main component that orchestrates the layout and functionality of the debugger interface.
- `DebuggerOutput`: A component for displaying the output of the code being debugged.
- `DebuggerInput`: A component for accepting user input code to be executed.
- `DebuggerControls`: A component providing buttons for various debugging actions.

These components are used together within the `Debugger` component to create a cohesive debugging interface.
