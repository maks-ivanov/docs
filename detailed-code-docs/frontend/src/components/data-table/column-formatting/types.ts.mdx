## High-level description

The target file, `types.ts`, defines TypeScript types and interfaces for managing column formatting options in a data table component. It specifies the available formatting options for different data types and extends the functionality of the TanStack Table library by integrating custom column formatting state, options, and instance methods.

## Code Structure

The main symbols in the code are interconnected as follows:
- `formatOptions`: A constant object that maps data types to their respective formatting options.
- `FormatOptions` and `FormatOption`: TypeScript types derived from `formatOptions` to represent possible formatting options.
- `ColumnFormattingState`, `ColumnFormattingTableState`, `ColumnFormattingOptions`, and `ColumnFormattingInstance`: Interfaces that define the state, options, and instance methods related to column formatting.
- Declaration merging is used to extend the TanStack Table's existing types with the custom column formatting types.

## Symbols

### `formatOptions`
#### Description
Defines the available formatting options for different data types in a data table. It is a constant object that maps each `DataType` to an array of strings representing the formatting options.

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| formatOptions | Record&lt;DataType, string[]&gt; | Maps data types to their respective formatting options. |

### `FormatOptions`
#### Description
A TypeScript type that represents all possible formatting options available across all data types.

### `FormatOption`
#### Description
A TypeScript type that represents a single formatting option from the `FormatOptions` type.

### `ColumnFormattingState`
#### Description
A TypeScript type that represents the state of column formatting, mapping column identifiers to their selected formatting options.

### `ColumnFormattingTableState`
#### Description
An interface that extends the table state to include column formatting state.

### `ColumnFormattingOptions`
#### Description
An interface that defines optional settings for enabling column formatting and handling changes to the column formatting state.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| enableColumnFormatting | boolean | Optional flag to enable column formatting. |
| onColumnFormattingChange | OnChangeFn&lt;ColumnFormattingState&gt; | Optional function to handle changes in column formatting state. |

### `ColumnFormattingInstance`
#### Description
An interface that defines methods for managing column formatting on a table column instance.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | FormatOption | The formatting option to set for a column. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| setColumnFormatting | void | Sets the formatting option for a column. |
| getColumnFormatting | FormatOption \| undefined | Retrieves the current formatting option for a column. |
| getCanFormat | boolean | Determines if a column can be formatted. |
| applyColumnFormatting | any | Applies the formatting to a column value. |
