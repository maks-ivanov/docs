## High-level description

The `Inputs.css` file defines the styling for various button components used in a frontend application. It specifies different color themes and states (such as hover and active) for buttons, using CSS variables for color values. The styles are designed to provide visual feedback and differentiation for buttons based on their intended use, such as neutral, alert, or destructive actions.

## Code Structure

The CSS file is structured around class selectors that apply styles to buttons based on their color theme and state. Each button type (e.g., gray, white, green, red, yellow, hint-green, and disabled) has specific styles for normal, hover, and active states. The styles are defined using CSS variables for consistency and maintainability.
