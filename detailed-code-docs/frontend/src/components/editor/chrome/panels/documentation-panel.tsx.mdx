## High-level description

The `DocumentationPanel` component in the target file is a React functional component that displays documentation related to a symbol in a code editor. It uses the `jotai` state management library to access the current documentation state and renders it as HTML. If no documentation is available, it displays an empty state message with an icon and description.

## Code Structure

The `DocumentationPanel` component relies on the `documentationAtom` from the `jotai` library to fetch the current documentation state. It uses the `renderHTML` function to safely render HTML content. The component also uses the `PanelEmptyState` component to display a message when no documentation is available.

## References

- `documentationAtom` from `frontend/src/core/documentation/state.ts`: Provides the state for the documentation content.
- `renderHTML` from `frontend/src/plugins/core/RenderHTML.tsx`: Renders HTML content safely.
- `PanelEmptyState` from `frontend/src/components/editor/chrome/panels/empty-state.tsx`: Displays a message when no documentation is available.

## Symbols

### `DocumentationPanel`
#### Description
The `DocumentationPanel` component is responsible for displaying documentation related to a symbol in a code editor. It checks if documentation is available and renders it using the `renderHTML` function. If no documentation is present, it displays a placeholder message using the `PanelEmptyState` component.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| None | N/A  | The component does not take any direct inputs but relies on the `documentationAtom` for state. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| JSX.Element | React Element | The rendered documentation or an empty state message. |

#### Internal Logic
1. The component uses the `useAtomValue` hook from `jotai` to access the `documentation` state from `documentationAtom`.
2. It checks if the `documentation` is available:
   - If `documentation` is not available, it returns the `PanelEmptyState` component with a title, description, and icon.
   - If `documentation` is available, it renders the HTML content using the `renderHTML` function within a styled `div`.
