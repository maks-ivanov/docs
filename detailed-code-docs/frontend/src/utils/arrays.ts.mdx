## High-level description

The `arrays.ts` file provides utility functions for manipulating arrays in TypeScript. These functions include operations for deleting, inserting, and comparing arrays, as well as a utility for zipping two arrays together. The file also defines a constant `Arrays` object that includes an empty array and a zip function, which ensures that two arrays are of the same length before combining them into an array of tuples.

## Code Structure

The main symbols in the code are utility functions that operate on arrays. These functions are generic, allowing them to work with arrays of any type. The `Arrays` object serves as a namespace for the `EMPTY` array and the `zip` function, which relies on the `invariant` function to ensure the arrays are of equal length.

## References

- The `invariant` function from `invariant.ts` is used to enforce conditions within the `zip` function.

#### Description
Compares two arrays for shallow equality, meaning it checks if they have the same length and corresponding elements are equal.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| a | T[] | The first array to compare. |
| b | T[] | The second array to compare. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| result | boolean | `true` if the arrays are shallowly equal, `false` otherwise. |

### `Arrays`
#### Description
An object containing utility properties and methods for array operations, including an empty array and a zip function.

#### Internal Logic
- `EMPTY`: An empty array of type `any[]`.
- `zip`: Combines two arrays of the same length into an array of tuples, using the `invariant` function to ensure the arrays are of equal length.

## Error Handling

The `zip` function uses the `invariant` function to throw an error if the two arrays being zipped are not of the same length, ensuring that the operation is only performed on arrays that can be paired element-wise.
