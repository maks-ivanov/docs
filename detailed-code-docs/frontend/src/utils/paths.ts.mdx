## High-level description

The `paths.ts` file provides utility functions and classes for handling file paths in a platform-independent manner. It includes methods to determine if a path is absolute, extract directory names, base names, and file extensions, and manipulate paths using a guessed delimiter based on the path format.

## Code Structure

The main symbols in the code are the `Paths` object and the `PathBuilder` class. The `Paths` object provides static utility functions for path manipulation, while the `PathBuilder` class offers methods to construct and deconstruct paths using a specified delimiter. The `FilePath` type is used to ensure type safety for path strings.
