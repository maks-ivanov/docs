## High-level description
This CSS file defines the styling for a custom Keycloak login theme named "trieve". It sets up a responsive design with light and dark mode support, custom fonts, and specific styling for login form elements and layout.

## Table of contents
- Font import
- Global styles
- Color scheme variables
- Media query for dark mode
- Body styles
- Input and button styles
- Login card styles
- Header and logo styles
- Button styles
- Miscellaneous styles (hr, signup, links)
- Social provider button styles

## Code Structure
The CSS file is structured with global styles at the top, followed by more specific element styling. It uses CSS variables for color theming and applies styles to both generic elements (like `body` and `input`) and specific Keycloak classes (like `.login-pf-page` and `#kc-header-wrapper`).

## Symbols

### `:root`
#### Description
Defines CSS variables for the color scheme used throughout the theme.

#### Internal Logic
- Sets default colors for light mode
- Uses a media query to override colors for dark mode preference

### `body`
#### Description
Sets the global styles for the body element.

#### Internal Logic
- Applies background color and text color using CSS variables
- Sets font family and size

### `input`, `button.pf-m-control::after`
#### Description
Styles form input elements and buttons.

#### Internal Logic
- Customizes border colors and styles
- Applies box shadows and border radius

### `.login-pf-page .card-pf`
#### Description
Styles the main login card container.

#### Internal Logic
- Sets border styles, padding, and box shadow
- Applies background color and text color

### `#kc-header-wrapper`
#### Description
Styles the header wrapper, including the logo.

#### Internal Logic
- Uses a pseudo-element to add the Trieve logo
- Sets font size and weight for the header text

### `.pf-c-button.pf-m-primary`
#### Description
Styles the primary action buttons (e.g., login button).

#### Internal Logic
- Sets background color, text color, and border styles
- Defines hover state styles

## Side Effects
The CSS applies global styles that may affect other elements if not properly scoped.
