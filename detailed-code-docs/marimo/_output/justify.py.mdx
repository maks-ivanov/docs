## High-level description

The `justify.py` file in the `marimo` library provides functions to justify HTML content in different alignments: center, left, and right. These functions convert an input object into an HTML representation and apply the specified alignment, returning an `Html` object that can be used in web-based applications or documents.

## Code Structure

The main symbols in the code are three functions: `center`, `left`, and `right`. Each of these functions takes an object as input, converts it to an HTML representation using the `as_html` function, and then applies the respective alignment method (`center`, `left`, or `right`) on the resulting `Html` object. The `Html` class, which is defined in the `hypertext.py` file, provides methods for these alignments.
