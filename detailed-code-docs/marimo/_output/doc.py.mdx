## High-level description

The `marimo/_output/doc.py` file defines a function `doc` that retrieves and formats documentation for a given object. If the object implements the `RichHelp` protocol, the documentation is rendered as markdown and returned as an `Html` object. Otherwise, the standard help documentation is printed to the console.

## Code Structure

The main function in this file is `doc`, which utilizes the `Html` class from `marimo._output.hypertext` and the `md` function from `marimo._output.md` to format and return documentation. It also uses the `mddoc` decorator from `marimo._output.rich_help` to enhance the function with additional documentation capabilities.

## References

- **`Html` class**: Used to encapsulate HTML content for rendering.
- **`md` function**: Converts markdown text into an `Html` object.
- **`mddoc` decorator**: Enhances functions or classes to implement the `RichHelp` protocol, which provides markdown-formatted documentation.

### `doc`
#### Description
The `doc` function retrieves documentation for a given object. If the object implements the `RichHelp` protocol, it returns the documentation as an `Html` object formatted in markdown. If not, it prints the standard help documentation to the console.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| obj | Any | The object for which documentation is to be retrieved. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| output | Optional[Html] | An `Html` object containing the documentation if the object implements `RichHelp`; otherwise, `None`. |

#### Internal Logic
1. The function checks if the object has a `_rich_help_` attribute, indicating it implements the `RichHelp` protocol.
2. If `_rich_help_` is present, it calls this method to get the documentation string.
3. The documentation string is then passed to the `md` function to convert it into an `Html` object.
4. If no documentation is available, a default message is converted to an `Html` object.
5. If the object does not implement `RichHelp`, the standard `help` function is called to print the documentation to the console.
