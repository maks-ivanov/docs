## Symbols

### `progress_bar`
#### Description
The `progress_bar` class provides a mechanism to iterate over a collection while displaying a progress bar in the UI. It can also be used as a context manager to manually update the progress bar.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| collection | Optional[Collection[S \| int]] | The collection to iterate over. |
| title | Optional[str] | The title to display on the progress bar. |
| subtitle | Optional[str] | The subtitle to display on the progress bar. |
| completion_title | Optional[str] | The title to display upon completion. |
| completion_subtitle | Optional[str] | The subtitle to display upon completion. |
| total | Optional[int] | The total number of items to iterate over. |
| show_rate | bool | Whether to show the rate of progress. |
| show_eta | bool | Whether to show the estimated time of completion. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| progress | ProgressBar | The progress bar object used to update the UI. |

#### Internal Logic
- Initializes a `ProgressBar` object with the provided parameters.
- Appends the progress bar to the output.
- Iterates over the collection, updating the progress bar with each item.
- Handles completion by updating the title and subtitle and closing the progress bar.
