## High-level description

The target file, `input.py`, is part of the Marimo UI plugin implementation. It defines a variety of interactive UI elements such as number pickers, sliders, checkboxes, radio buttons, text inputs, and more. These elements are designed to be used in web applications to capture user input and provide interactive features. Each UI element is implemented as a class that extends a base class `UIElement`, providing specific functionality and attributes for different types of input components.

## Code Structure

The main symbols in the code are classes that represent different UI elements. Each class inherits from the `UIElement` base class, which provides common functionality for handling UI interactions. The classes are designed to encapsulate the logic for specific types of input components, such as number pickers, sliders, checkboxes, etc. Each class has methods for initialization, value conversion, and interaction handling.

### `number`
#### Description
The `number` class represents a number picker UI element that allows users to select a number within a specified range. It provides attributes for the start, stop, and step values of the range, as well as an optional callback for when the value changes.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| start | float | The minimum value of the interval. |
| stop | float | The maximum value of the interval. |
| step | Optional[float] | The number increment. |
| value | Optional[float] | Default value. |
| debounce | bool | Whether to debounce value updates from the frontend. |
| label | str | Text label for the element. |
| on_change | Optional[Callable[[Optional[Numeric]], None]] | Callback for value changes. |
| full_width | bool | Whether the input should take up the full width of its container. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| value | Optional[Numeric] | The current value of the number picker. |

#### Internal Logic
- Validates the bounds and default value.
- Initializes the UI element with the specified attributes.
- Provides a static method `from_series` to create a number picker from a dataframe series.
