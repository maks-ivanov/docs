## High-level description

The `tabs` class in the `marimo/_plugins/ui/_impl/tabs.py` file is a UI component that allows for the creation of tabbed interfaces in applications. It provides functionality to display content in a tabbed view, control which tab is selected, and optionally load tab content lazily. This class is part of the Marimo UI framework, which is designed to facilitate the creation of interactive and dynamic user interfaces.

## Code Structure

The `tabs` class inherits from the `UIElement` class, which is a generic class for UI components that have a value and can interact with the frontend. The `tabs` class uses several utility functions and classes from the Marimo framework, such as `as_html`, `md`, and `lazy_ui`, to render content and manage lazy loading.

### `_convert_value`
#### Description
This method converts a string value representing the selected tab index into the corresponding tab name.

#### Inputs
| Name  | Type   | Description                        |
|:------|:-------|:-----------------------------------|
| `value` | `str` | The index of the selected tab as a string. |

#### Outputs
| Name   | Type   | Description                        |
|:-------|:-------|:-----------------------------------|
| `tab_name` | `str` | The name of the selected tab.   |

#### Internal Logic
- If the `value` is empty, it returns the name of the first tab.
- Otherwise, it converts the `value` to an integer index and returns the corresponding tab name from `_tab_keys`.
