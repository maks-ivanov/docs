## High-level description
The `Tooltip` component is a reusable UI element that displays a tooltip text when hovering over a target element. It allows customization of the tooltip's direction (top, bottom, left, right) and provides a simple interface for specifying the tooltip content and the target element.

## Symbols

### `Tooltip`
#### Description
This component renders a tooltip that appears when hovering over a target element. The tooltip's direction and content are customizable through props.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| body | JSX.Element | The target element to trigger the tooltip |
| tooltipText | string | The text content of the tooltip |
| direction | TooltipDirection | The direction of the tooltip relative to the target element (default: 'bottom') |

#### Outputs
The component renders a `div` element containing the target element and a conditionally rendered tooltip.

#### Internal Logic
1. The component uses the `createSignal` hook to manage the visibility of the tooltip (`show`).
2. The `createMemo` hook is used to determine the tooltip's direction, defaulting to 'bottom' if not provided.
3. The target element is wrapped in a `div` with event listeners for `onMouseEnter` and `onMouseLeave` to control the tooltip's visibility.
4. The `Show` component conditionally renders the tooltip based on the `show` signal.
5. The tooltip's position is determined using dynamic class names based on the `direction` memo.
