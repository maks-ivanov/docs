## Symbols

### `cn`
#### Description
This function merges Tailwind CSS classes with other CSS classes using the `clsx` and `tailwind-merge` libraries. It takes a variable number of arguments, each representing a CSS class or an array of CSS classes. The function first uses `clsx` to conditionally join the classes, then applies `tailwind-merge` to merge the Tailwind classes.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ...args | `ClassValue[]` | An array of CSS classes or arrays of CSS classes to be merged. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| mergedClasses | `string` | A string containing the merged CSS classes. |

#### Internal Logic
1. The function uses the spread syntax (`...args`) to collect all arguments into an array.
2. It calls the `clsx` function with the array of arguments. `clsx` conditionally joins the classes based on their truthiness.
3. The result of `clsx` is then passed to the `twMerge` function from the `tailwind-merge` library. This function merges the Tailwind classes and returns the final merged class string.
