## High-level description
The Sidebar component is a part of a chat application's user interface. It displays a list of topics, allows users to create, edit, and delete topics, and provides navigation and settings options.

## Code Structure
The Sidebar component is the main export of this file. It uses various imported components and icons to create a sidebar interface. The component manages its own state for editing topics and displaying settings. It also interacts with the UserContext to access and modify user-related data.

## Symbols

### `Sidebar`
#### Description
The main component that renders the sidebar interface.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | SidebarProps | Contains functions and data for managing topics and sidebar state |

#### Internal Logic
1. Initializes state variables for editing topics and settings modal
2. Fetches star count from GitHub API
3. Renders the sidebar content, including:
   - New topic button
   - List of existing topics
   - Organization and dataset selection
   - Navigation links
   - Settings button

### `submitEditText`
#### Description
Submits an edited topic name to the server.

#### Internal Logic
1. Retrieves the current dataset
2. Sends a PUT request to update the topic name
3. Refreshes the topics list on success
4. Displays an error toast on failure

### `deleteSelected`
#### Description
Deletes the currently selected topic.

#### Internal Logic
1. Retrieves the current dataset
2. Sends a DELETE request to remove the topic
3. Refreshes the topics list on success
4. Displays an error toast on failure

## Side Effects
