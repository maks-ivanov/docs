## High-level description
The `DatasetSelectionList` component is a reusable React component that renders a list of selectable datasets. It allows users to choose a dataset from the provided list and updates the current dataset in the user context when selected.

## Code Structure
The main symbol in this code is the `DatasetSelectionList` component. It uses the `UserContext` to access and update the current dataset. The component receives props including an `onSelect` callback and an array of `datasets`.

## Symbols

### DatasetSelectionList
#### Description
A functional component that renders a list of selectable datasets using the `For` component from Solid.js.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | DatasetSelectionListProps | Contains the onSelect callback and datasets array |

#### Internal Logic
1. Retrieves the user context using `useContext`.
2. Renders a list of datasets using the `For` component.
3. For each dataset, renders a `MenuItem` with the dataset name and a checkmark for the currently selected dataset.
4. Handles click events on dataset items to update the current dataset and call the `onSelect` callback.
