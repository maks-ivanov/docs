## High-level description
This code is the entry point for a chat application built with SolidJS. It sets up Sentry for error tracking in production, renders the main Chat component wrapped in a UserContextWrapper, and mounts the application to the DOM.

## Table of contents
- Imports
- Sentry initialization
- Root element selection
- Application rendering

## Code Structure
The code imports necessary dependencies and components, initializes Sentry for non-development environments, and renders the main application structure using SolidJS.

## References
- `Chat` component from "./pages/chat"
- `UserContextWrapper` component from "./components/contexts/UserContext"

## Symbols

### Sentry Initialization
#### Description
Initializes Sentry for error tracking and performance monitoring in non-development environments.

#### Internal Logic
- Checks if the application is not in development mode
- Sets up Sentry with various configurations including DSN, integrations, and sampling rates

### Root Element Selection
#### Description
Selects the DOM element with the id "root" to serve as the mounting point for the application.

### Application Rendering
#### Description
Renders the main application structure using SolidJS's render function.

#### Internal Logic
- Wraps the `Chat` component with `UserContextWrapper`
- Mounts the application to the selected root element
