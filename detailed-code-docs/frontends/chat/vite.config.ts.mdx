## High-level description
This file configures Vite for a SolidJS project. It sets up the necessary plugins to support SolidJS development and runtime environment variables.

## Table of contents
- Vite configuration
- Plugin setup

## Code Structure
The code exports a default Vite configuration object created using `defineConfig`. This configuration includes two plugins: `solid()` and `runtimeEnv()`.
