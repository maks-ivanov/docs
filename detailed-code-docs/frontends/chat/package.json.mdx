## High-level description
This `package.json` file defines the configuration, dependencies, and scripts for a chat frontend application built with SolidJS and Vite. It includes various development tools, UI libraries, and utilities for building a modern, interactive chat interface.

## Table of contents
- Scripts
- Dependencies

## Code Structure
The file is structured as a standard `package.json`, containing metadata about the project, scripts for development and building, and lists of production and development dependencies.

## Symbols

### Scripts
#### Description
Defines various npm scripts for development, building, and linting the project.

#### Internal Logic
- `dev`: Starts the Vite development server on port 5175
- `build`: Compiles TypeScript and builds the project for production
- `preview`: Previews the built project
- `lint`: Runs Prettier and ESLint to format and fix code issues
- `lint:only`: Runs ESLint only
- `lint:CI`: Runs ESLint and outputs results to a JSON file for CI environments

### Dependencies
#### Description
Lists the production dependencies required for the chat application.

#### Internal Logic
Key dependencies include:
- `solid-js`: The core SolidJS library
- `@solidjs/router`: Routing library for SolidJS
- `solid-markdown`: Markdown rendering for SolidJS
- `@sentry/browser`: Error tracking with Sentry
- Various accessibility-related packages (`@solid-aria/*`)
- Markdown-related packages (`remark-*`, `rehype-sanitize`)
- `sanitize-html`: HTML sanitization library
- `tailwind-*`: Tailwind CSS related utilities

### DevDependencies
#### Description
Lists the development dependencies used for building, testing, and linting the project.

#### Internal Logic
Key dev dependencies include:
- `vite` and related plugins: For building and bundling the application
- `typescript`: For TypeScript support
- `eslint` and related plugins: For code linting
- `prettier`: For code formatting
- `tailwindcss` and related tools: For styling
- `autoprefixer` and `postcss`: For CSS processing
