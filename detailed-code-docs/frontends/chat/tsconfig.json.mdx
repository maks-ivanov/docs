## High-level description
This file is a TypeScript configuration file (`tsconfig.json`) for the chat frontend of the project. It extends a base configuration and specifies compiler options and source file inclusion for the TypeScript compiler.

## Table of contents
- Base configuration extension
- Compiler options
- Source file inclusion

## Code Structure
The configuration is structured as a JSON object with three main properties: `extends`, `compilerOptions`, and `include`.

## Symbols

### `extends`
#### Description
Specifies the base configuration file to extend from.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| extends | string | Path to the base configuration file |

### `compilerOptions`
#### Description
Defines compiler options specific to this project.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| noUnusedLocals | boolean | Disables the error for unused local variables |

### `include`
#### Description
Specifies which files should be included in the compilation.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| include | array of strings | Patterns to match files for inclusion |

## Dependencies

## Configuration
| Option | Type | Default | Description |
|:-------|:-----|:--------|:------------|
