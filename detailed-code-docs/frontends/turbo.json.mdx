## High-level description
This `turbo.json` file is a configuration file for Turborepo, a high-performance build system for JavaScript and TypeScript monorepos. It defines the build pipeline and task configurations for the project, specifying how different tasks should be executed and cached.

## Table of contents
- Schema definition
- UI configuration
- Task configurations
  - build
  - lint
  - lint:CI
  - dev

## Code Structure
The configuration is structured as a JSON object with top-level keys for schema, UI, and tasks. The tasks object contains individual task configurations.

## Symbols

### `$schema`
#### Description
Specifies the JSON schema for the Turborepo configuration file, ensuring proper validation and IDE support.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| $schema | string | URL of the Turborepo JSON schema |

### `ui`
#### Description
Defines the user interface type for Turborepo, set to "tui" (Text User Interface).

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| ui | string | Type of user interface ("tui" for Text User Interface) |

### `tasks`
#### Description
Defines the configuration for various tasks in the build pipeline.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| tasks | object | Object containing task configurations |

### `tasks.build`
#### Description
Configures the build task, specifying the output directory for build artifacts.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| outputs | array of strings | Glob patterns for build output directories |

### `tasks.lint`
#### Description
Configures the lint task, disabling caching for this task.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cache | boolean | Whether to cache the task results (set to false) |

### `tasks.lint:CI`
#### Description
Configures a separate lint task for CI environments, also with caching disabled.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| cache | boolean | Whether to cache the task results (set to false) |

### `tasks.dev`
#### Description
Configures the development task, setting it as persistent and disabling caching.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| persistent | boolean | Whether the task should run persistently |
| cache | boolean | Whether to cache the task results (set to false) |
