## High-level description
This code defines the Vite configuration for a SolidJS project, specifically for the "dashboard" frontend. It sets up plugins for SolidJS, Markdown processing, and runtime environment variables.

## Table of contents
- Plugins

## Symbols
### `defineConfig`
#### Description
This function, imported from "vite", is used to define the Vite configuration for the project. It takes an object with various configuration options.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | object | An object containing the Vite configuration options. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | object | Returns the Vite configuration object. |

### `solid`
#### Description
This function, imported from "vite-plugin-solid", is a Vite plugin that provides SolidJS support. It is configured to handle ".mdx" and ".md" file extensions.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| options | object | An object containing options for the SolidJS plugin. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| plugin | object | Returns the SolidJS plugin object. |

### `runtimeEnv`
#### Description
This function, imported from "vite-plugin-runtime-env", is a Vite plugin that enables the use of runtime environment variables in the project.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| options | object | An object containing options for the runtime environment variables plugin. |

#### Outputs
| Name | Type | Description |

| plugin | object | Returns the runtime environment variables plugin object. |
