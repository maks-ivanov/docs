## High-level description
The `DatasetName` component displays the name of the current dataset. It retrieves the dataset name from the `DatasetContext` and renders it within an `h3` tag.

## Table of contents

## References

## Symbols
### Symbol Name
`DatasetName`

#### Description
This component retrieves the current dataset name from the `DatasetContext` and displays it within an `h3` tag with styling.

#### Inputs
None

#### Outputs
JSX Element: An `h3` element containing the current dataset name.

#### Internal Logic
1. Retrieves the `datasetContext` using `useContext(DatasetContext)`.
2. Creates a memo `curDatasetName` that:
    - Accesses the `dataset` from the `datasetContext`.
    - If the `dataset` exists, returns its `name` property.
    - Otherwise, returns `null`.
3. Renders an `h3` element containing the value of `curDatasetName()`.

## Dependencies
- `solid-js`: Used for creating memos, using contexts, and JSX elements.
- `DatasetContext`: Provides access to the current dataset.
