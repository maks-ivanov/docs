## High-level description
The `DatasetTabs` component renders a set of navigation tabs for navigating between different sections of a specific dataset in the dashboard. It dynamically constructs the links based on the current dataset ID and selected organization.

## Code Structure
The `DatasetTabs` component retrieves context and URL parameters to dynamically generate navigation links. It uses the `UserContext` to access the selected organization ID and the `useParams` hook to get the current dataset ID. These values are used to construct the URLs for each tab.

## References

## Symbols

### Symbol Name
`DatasetTabs`

#### Description
This functional component renders a set of navigation tabs for different sections of a dataset in the dashboard.

#### Inputs
This component does not accept any explicit input props.

#### Outputs
Returns a `&lt;div&gt;` element containing the rendered navigation tabs.

#### Internal Logic
1. **Retrieves Context and Params:**
   - Uses `useContext(UserContext)` to access the `UserContext` and retrieve the `selectedOrganizationId`.
   - Uses `useParams()` to get the `dataset_id` from the URL parameters.

2. **Creates Current Organization ID Memo:**
   - Uses `createMemo` to create a memoized value for the `currentOrgId`, which is derived from the `selectedOrganizationId` from the `UserContext`.

3. **Renders Tabs:**
   - Renders a `&lt;div&gt;` element containing four `&lt;A&gt;` (anchor) components, each representing a tab: "Start", "Events", "API Keys", and "Settings".
   - Each `&lt;A&gt;` component's `href` attribute is dynamically constructed using the `dataset_id` and `currentOrgId`.
   - The `activeClass` prop is used to visually highlight the currently active tab based on the URL.

## Dependencies
- `@solidjs/router` - Used for navigation and URL manipulation (`A`, `useParams`).
