## High-level description
The `BuildingSomething` component displays a styled message encouraging users to share their projects on Discord or Matrix.

## Table of contents
- JSX Structure

## Symbols

### `BuildingSomething`
#### Description
This functional component renders a styled `div` containing a message that prompts users to share their projects. It includes links to Discord and Matrix.

#### Inputs
This component accepts no props.

#### Outputs
Returns a JSX structure representing a styled message.

```

  &lt;BiRegularInfoCircle class="h-5 w-5 text-blue-400" /&gt;
  &lt;p class="text-sm text-blue-700"&gt;
    Building something? Share in our{" "}
    &lt;a class="underline" href="https://discord.gg/s4CX3vczyn"&gt;
      Discord
    &lt;/a&gt;{" "}
```
