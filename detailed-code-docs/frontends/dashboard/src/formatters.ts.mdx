## High-level description
This code defines a set of formatting utility functions for currency, bytes, numbers, and dates. These formatters are intended for use in a web application, likely a dashboard, to present data in a user-friendly way.

## Table of contents
- `usdFormatter`
- `formatBytesDecimal`
- `numberFormatter`
- `formatDate`

## Symbols

### `usdFormatter`
#### Description
Creates a new `Intl.NumberFormat` instance configured to format numbers as US dollars.

#### Inputs
None

#### Outputs
An `Intl.NumberFormat` instance.

#### Internal Logic
Uses the `Intl.NumberFormat` constructor with the 'en-US' locale and 'currency' style, specifying 'USD' as the currency.

### `formatBytesDecimal`
#### Description
Formats a given number of bytes into a human-readable string representation in megabytes (MB) or gigabytes (GB).

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| bytes | number | The number of bytes to format. |
| dm | number | (Optional) The number of decimal places to round to. Defaults to 2. |

#### Outputs
A string representation of the byte size in MB or GB.

#### Internal Logic
- If `bytes` is 0, returns "0 Bytes".
- If `bytes` is greater than 1e8 (100 MB), divides by 1e9 (1 GB) and rounds to `dm` decimal places, appending " GB".
- Otherwise, divides by 1e6 (1 MB) and rounds to `dm` decimal places, appending " MB".

### `numberFormatter`
#### Description
Creates a new `Intl.NumberFormat` instance configured to format numbers according to the 'en-US' locale conventions.

#### Inputs
None

#### Outputs
An `Intl.NumberFormat` instance.

#### Internal Logic
Uses the `Intl.NumberFormat` constructor with the 'en-US' locale.

### `formatDate`
#### Description
Formats a given `Date` object into a human-readable date string according to the 'en-US' locale conventions.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| date | Date | The `Date` object to format. |

#### Outputs
A string representation of the date.

#### Internal Logic
Creates a new `Intl.DateTimeFormat` instance with the 'en-US' locale and calls its `format` method with the provided `date`.
