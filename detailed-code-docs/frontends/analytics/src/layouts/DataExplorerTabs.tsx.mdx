## High-level description
The `DataExplorerTabs` component renders a set of navigation tabs for the data explorer section of the application. It highlights the currently active tab based on the URL pathname and uses a custom navigation function to preserve query parameters.

## Symbols

### Symbol Name
#### `DataExplorerTabs`
#### Description
This component renders a navigation bar with tabs for different data explorer pages. It highlights the active tab based on the current URL and uses a custom navigation function to preserve query parameters when switching between tabs.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| children | JSX.Element | The child elements to be rendered within the tab content area |

#### Outputs
Renders the navigation tabs and the provided child elements within a container.

#### Internal Logic
1. **Defines pages:** An array `pages` is defined with the name and path of each tab.
2. **Uses hooks:** It utilizes the `useBetterNav` and `usePathname` hooks to handle navigation and determine the active tab.
3. **Renders tabs:** It iterates through the `pages` array and renders an `&lt;A&gt;` element for each tab, applying a highlight class if the tab's path matches the current pathname.
4. **Handles clicks:** When a tab is clicked, it prevents default behavior and uses the `betterNav` function to navigate to the tab's path while preserving query parameters.
5. **Renders children:** Finally, it renders the provided `children` within a container below the tabs.
