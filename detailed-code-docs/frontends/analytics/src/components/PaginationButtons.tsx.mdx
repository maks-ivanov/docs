## High-level description
The `PaginationButtons` component renders a simple pagination control with buttons for navigating to the previous and next pages. It displays the current page number and disables the navigation buttons when at the first or last page.

## Symbols

### Symbol Name
`PaginationButtons`

#### Description
A functional component that renders pagination buttons. It uses the `usePagination` hook to manage the current page and determine whether navigation buttons should be disabled.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | `PaginationButtonsProps` | An object containing the following properties:
|  props.pages | `ReturnType&lt;typeof usePagination&gt;` | An object returned from the `usePagination` hook, providing access to pagination state and functions.
|  props.size | `number` (optional) | The size of the navigation icons. Defaults to 24.

#### Outputs
- JSX: A `&lt;div&gt;` element containing the pagination buttons and current page number.

#### Internal Logic
1. Renders a `&lt;div&gt;` container with flexbox styling for the buttons.
2. Renders a button for navigating to the previous page. The button is disabled if the current page is 1.
3. Renders a `&lt;div&gt;` element displaying the current page number.
4. Renders a button for navigating to the next page. The button is disabled if the current page is the last page.
5. The navigation buttons use the `AiFillCaretLeft` icon with rotation and opacity styling to indicate direction and disabled state.
