## High-level description
The `useBetterNav` function is a custom navigation hook for SolidJS applications. It provides an enhanced navigation function that preserves existing query parameters while navigating to a new route.

## Symbols

### `useBetterNav`
#### Description
This function is a custom hook that enhances the `navigate` function from `@solidjs/router`. It creates a navigation function that preserves existing query parameters from the current URL and appends them to the new route.

#### Inputs
This function takes no direct inputs. However, it uses the following SolidJS hooks internally:
- `useSearchParams`: To access the current URL's query parameters.
- `useNavigate`: To perform the actual navigation.

#### Outputs
Returns a function that takes a single argument:
- `pathname`: `string` representing the pathname of the new route to navigate to.

#### Internal Logic
1. Retrieves the current query parameters using `useSearchParams`.
2. Constructs a new `URLSearchParams` object from the retrieved query parameters.
3. Generates a query string from the `URLSearchParams` object.
4. Uses the `navigate` function from `useNavigate` to navigate to the specified `pathname`, appending the generated query string if it exists.

#### Example Usage
```typescript
const navigate = useBetterNav();

// Navigate to '/new-route' while preserving query parameters
navigate('/new-route'); 
```
