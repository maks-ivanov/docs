## High-level description
This code defines the Vite configuration for an analytics frontend project built with SolidJS. It leverages Vite plugins for SolidJS integration and runtime environment variables.

## Table of contents
- Plugins

## Symbols

### `defineConfig`
#### Description
This function, imported from Vite, defines the configuration for a Vite project. It takes an object with various configuration options.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | object | An object containing Vite configuration options. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| config | object | Returns a Vite configuration object. |

### `plugins`
#### Description
This option within the Vite configuration object defines an array of Vite plugins to be used in the project.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| plugins | array | An array of Vite plugins. |

#### Internal Logic
- `solid()`: This plugin, imported from `vite-plugin-solid`, provides first-class SolidJS support for Vite projects.
- `runtimeEnv()`: This plugin, imported from `vite-plugin-runtime-env`, enables the use of runtime environment variables in the project.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| vite | Provides the core build tool functionality. |
| vite-plugin-solid | Enables SolidJS support in Vite. |
| vite-plugin-runtime-env | Allows using runtime environment variables. |
