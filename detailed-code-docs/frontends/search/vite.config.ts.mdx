## High-level description
This file configures Vite for a Solid.js project with runtime environment variable support. It uses the `defineConfig` function from Vite to set up the build process with two plugins: `solid` for Solid.js support and `runtimeEnv` for runtime environment variables.

### Configuration Object
#### Description
The configuration object passed to `defineConfig` that sets up the Vite build process.

#### Internal Logic
The configuration object includes a `plugins` array with two plugins:
1. `solid()`: Initializes the Solid.js plugin for Vite.
2. `runtimeEnv()`: Initializes the runtime environment variables plugin for Vite.
