## High-level description
The `InputRowsForm` component renders a form with customizable input rows, a cancel button, and a create button. It allows users to input data and triggers actions upon button clicks.

## Symbols

### `InputGroup`
#### Description
Defines the structure of an input group within the form.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| label | `string` | Label displayed above the input field |
| inputValue | `Accessor&lt;string&gt;` | Accessor function to get the current input value |
| setInputValue | `(value: string) =&gt; void` | Function to update the input value |
| type | `"input" \| "textarea"` | Type of input field, defaults to "input" |
| component | `JSXElement` | Custom component to render instead of default input fields |

### `InputRowsFormProps`
#### Description
Defines the properties for the `InputRowsForm` component.

#### Inputs

|:-----|:-----|:------------|

| borderColor | `string` | Optional: Custom border color for the form container |
| buttonClass | `string` | Optional: Custom CSS class for the buttons |

### `InputRowsForm`
#### Description
Renders a form with customizable input rows, a cancel button, and a create button.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | `InputRowsFormProps` | Properties of the component |

#### Internal Logic
1. Iterates through the `inputGroups` array.
2. For each `InputGroup`, renders either a default input field ("input" or "textarea" based on `type`) or a custom component provided in `component`.
3. Renders a cancel button and a create button with the provided text and click handlers.
