## High-level description
The `PaginationController` component renders a pagination control for navigating through multiple pages of results. It provides buttons for moving to the previous and next pages, as well as a dynamic range of page numbers centered around the current page.

- `createArrayWithCenteredRange` function

## Symbols

### `createArrayWithCenteredRange`
#### Description
This function generates an array of page numbers to be displayed in the pagination control. It ensures that the current page is centered within the displayed range, unless the current page is close to the beginning or end of the total pages.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| center | number | The page number to center the range around. |
| range | number | The desired number of page numbers to display in the control. |

#### Outputs
| Name | Type | Description |
|:-----|:-----|:------------|
| array | number[] | An array of page numbers to be displayed in the pagination control. |

#### Internal Logic
1. Calculates the number of indices to include before and after the `center` index.
2. If the `center` is close to the beginning or end of the range, it displays a sequential range from 1 or from `totalPages - range` respectively.
3. Otherwise, it calculates the starting page number based on the `center` and `indicesBeforeCenter`.
4. Iterates `range` times, pushing the current page number to the `array` and incrementing it.

### `PaginationController`
#### Description
This component renders the pagination control UI, including the previous/next buttons and the dynamic page number range. It uses the `createArrayWithCenteredRange` function to determine the page numbers to display.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| setPage | (page: number) =&gt; void | Callback function to update the current page state. |
| page | number | The current page number. |
| totalPages | number | The total number of pages available. |

#### Outputs
Renders the pagination control UI.

#### Internal Logic
1. Renders a "previous page" button if the current page is not the first page.
2. Uses `createArrayWithCenteredRange` to generate an array of page numbers to display.
3. Iterates through the page numbers, rendering a button for each one. The button's styling is dependent on whether it represents the current page or not.
4. Renders a "next page" button if the current page is not the last page.
