## High-level description
The `ViewChunk.tsx` component renders a page for viewing a single chunk of data. It fetches the chunk data based on the provided `chunkId` and displays it using the `SingleChunkPage` component. 

## Table of contents
- Imports
- ViewChunk component

## Symbols

### Symbol Name: `ViewChunk`
#### Description:
This functional component is responsible for rendering the view of a single chunk. It retrieves the chunk ID from the route parameters, fetches the chunk data, and passes it to the `SingleChunkPage` component for display. It also renders the `Footer` component at the bottom of the page.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | any | Component properties, including `params` object containing the `id` of the chunk to view. |

#### Outputs
JSX Element representing the rendered single chunk view.

#### Internal Logic
1. Renders the `SearchLayout` component.
2. Renders the `SingleChunkPage` component, passing the `chunkId` extracted from the route parameters (`props.params.id`) and a default `ResultChunk` object.
3. Renders an empty `div` with classes `flex-1` for spacing.
4. Renders the `Footer` component.

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| Footer | Provides the footer content for the page. |
| SearchLayout | Provides the layout structure for the search page. |
| SingleChunkPage | Renders the content of a single chunk. |
