## High-level description
The `EditChunk.tsx` component provides the user interface for editing an existing document chunk within a specific dataset. It fetches the chunk data, displays an editable form with the chunk's content and metadata, and handles the update request to the backend.

## Symbols

### `EditChunk` component
#### Description
This component renders the page for editing an existing document chunk. It fetches the chunk data based on the provided `chunkId` and passes it to the `EditChunkPageForm` component for editing.

#### Inputs
| Name | Type | Description |
|:-----|:-----|:------------|
| props | any | Component properties, including `params` object containing the `id` of the chunk to edit. |

#### Outputs
JSX.Element representing the Edit Chunk page.

#### Internal Logic
1. Renders the `SearchLayout` component to provide the basic page structure.
2. Displays a heading and a logo.
3. Renders the `EditChunkPageForm` component, passing the `chunkId` extracted from the `props` and an initial empty `defaultResultChunk` object.
4. Renders the `Footer` component.
