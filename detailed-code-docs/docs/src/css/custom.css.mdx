## High-level description
This CSS file defines global styles for a website using the Infima CSS framework. It sets custom color variables for the primary color scheme and adjusts the code font size. The file also includes a separate color scheme for dark mode.

## Symbols

### `:root`
#### Description
The `:root` pseudo-class defines custom CSS variables (also known as CSS custom properties) that can be used throughout the stylesheet. These variables set the primary color scheme and code font size for the light mode of the website.

#### Internal Logic
- Defines various shades of the primary color using the `--ifm-color-primary` prefix
- Sets the code font size to 95% of the default size
- Defines the background color for highlighted code lines

### `[data-theme='dark']`
#### Description
This attribute selector targets elements with the `data-theme` attribute set to 'dark'. It redefines the custom CSS variables for the dark mode of the website.

#### Internal Logic
- Overrides the primary color scheme with lighter shades suitable for dark mode
- Adjusts the background color for highlighted code lines in dark mode
