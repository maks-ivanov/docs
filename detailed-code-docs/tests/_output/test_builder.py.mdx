## High-level description

The target file `tests/_output/test_builder.py` contains a suite of unit tests for the HTML element generation functions provided by the `_HTMLBuilder` class in the `marimo._output.builder` module. These tests verify that the HTML elements are correctly constructed with the specified attributes and content.

## Code Structure

The main symbols in the code are the test functions, each of which corresponds to a method in the `_HTMLBuilder` class. These test functions use assertions to verify that the output of the HTML builder methods matches the expected HTML strings. The `_join_params` function is also tested to ensure it correctly formats HTML attributes.
