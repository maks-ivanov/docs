## High-level description
This CSS file (style.css) defines the styling for a treasure hunt web application. It sets the overall layout, typography, and visual appearance of various elements, ensuring a clean and modern design with a focus on readability and user experience.

## Symbols

### body
#### Description
Defines the styling for the entire body of the HTML document.

#### Internal Logic
- Sets the background color to a light gray (#f5f5f5)
- Uses 'Roboto' sans-serif font for text
- Creates a flexbox layout to center content vertically and horizontally
- Ensures the body takes up at least the full viewport height

### .container
#### Description
Styles the main container of the application.

#### Internal Logic
- Sets a maximum width of 500 pixels
- Sets the width to 60% of its parent container
- Adds horizontal padding of 10 pixels

### .input, .button
#### Description
Removes border radius from input fields and buttons.

#### Internal Logic
- Sets border-radius to 0 to create square corners

### .box
#### Description
Styles box elements used for content sections.

#### Internal Logic
- Adds a subtle box shadow for depth
- Removes border radius for square corners
- Adds 30 pixels of padding inside the box

### .markdown-body
#### Description
Styles the container for markdown-rendered content.

#### Internal Logic
- Sets the box-sizing to border-box
- Matches the background color with the body
- Adds a subtle box shadow
- Removes border radius for square corners
- Adds 30 pixels of padding inside the container

### .gpt-logo
#### Description
Styles the container for the GPT logo.

#### Internal Logic
- Takes up 100% of its parent's height and width
- Uses flexbox to center the logo both vertically and horizontally

### .gpt-logo img
#### Description
Styles the GPT logo image.

#### Internal Logic
- Sets both width and height to 15% of its container

## Dependencies
| Dependency | Purpose |
|:-----------|:--------|
| Roboto font | Used as the primary font for the application |

## References
This CSS file is referenced in the `index.html` file:
```html
&lt;link rel="stylesheet" href="{{ url_for('static', filename='css/style.css') }}"&gt;
```

The styling defined here complements the structure laid out in the HTML file, particularly affecting elements with classes such as `.container`, `.box`, `.input`, `.button`, and `.markdown-body`.
